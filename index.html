BibleModel

Welcome to my current research project found in http://github.com/MichaelKentBurns/BibleModel
This is a clone of https://github.com/MichaelKentBurns/BibleModel.git

Notes

  1. BibleModel uses a preferences json file named BibleModel-preferences.json to allow the user to customize the user's experience. There is a default copy of this file in the root of this package. It is my intention that this project will also support a collection of user accounts. But, I'm not ready to do that yet. In the mean time, you can "login" by opening the preferences page (first on the list below) in order to view the default set of preferences. When you do that the first time in your browser, a copy of those preferences will be stored in your browser's LocalStorage. The other pages shown below will by default look in the local storage to find it.

Currently available pages from this project

  1. Your preferences for BibleModel (preferences.html)
  2. Versions of the Bible (versions.html)
  3. Abbreviated names for the books of the Bible (book-abbreviationss.html)
  4. Books of the Bible (books.html)
  5. One Book of the Bible (book.html)
  6. One Book of the Bible with Contents (bookWithContents.html)
  7. One Book of the Bible with Cross References (bookWithXref.html)
  8. Create or edit notes. [Work in progress](Notes.html)
  9. Save/View Notes data [Work in progress](NotesData.html)
  10. Notes Manager [Work in progress](NotesManager.html)
  11. This index, but always fetched from REST server. (indexREST.html)
