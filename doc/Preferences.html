BibleModel preferences settings.

The BibleModel program (or Node.js package) uses a preferences file in a JSON document, normally located in the root of the package in a file called BibleModel-preferences.json.

This page describes those settings.

Details each preference:

  • Instructions is simply a description because JSON data can't have comments in the usual sense.
  • Documentation is the location of this web page.
  • accessDataBy-options accessDataBy can be either:
    • file
      • access files stored in the MyBiblePath directory.
    • REST
      • access files using the REST API that is built into this package. After initialization, the program becomes an http server which surfaces a REST interface. Accessing the '/' endpoint of that interface will elicit a list of the other endpoints that are available.
  • accessDataBy determines how the html files in DevUI folder access the data in the JSON document.
